Installation
===========
PlanktonPy requires Python >3.6 and can be installed with pip.

Unix
^^^^
Open Terminal and  install with pip::
    
    pip install planktonpy

MacOs
^^^^^
Open Terminal and  install with pip::
    
    Python3 -m pip install planktonpy
    
    
Windows
^^^^^
Open Terminal and  install with pip::
    
    Python3 -m pip install planktonpy
